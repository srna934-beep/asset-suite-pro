## Technical notes
- New tables: `profiles`, `user_roles`, `notifications`. All with GRANTs + RLS + `has_role`-based policies.
- Auth gate uses integration-managed `src/routes/_authenticated/route.tsx`.
- Sidebar refactored to use shadcn `Sidebar` + `SidebarProvider` so the hamburger `SidebarTrigger` works on mobile.
- Google OAuth via `lovable.auth.signInWithOAuth("google", ...)` + `supabase--configure_social_auth`.
- Notifications generated by a DB function + cron-style refresh on dashboard load (client-side `ensure` call).
